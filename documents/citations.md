# Citations & Data Sources

## Primary Dataset

Sleep Efficiency Datast
Source: Kaggle
Link : https://www.kaggle.com/datasets/equilibriumm/sleep-efficiency/discussion/498373

This dataset contains 1,000 participants and includes variables such as:

- Age
- Gender
- Sleep Duration
- Sleep Efficiency
- REM, Deep, and Light Sleep Percentages
- Awakenings
- Caffeine Consumption (mg)
- Alcohol Consumption (oz)
- Smoking Status
- Exercise Frequency

This dataset file ('sleepstudy.csv') is excluded from version control using `.gitignore` to maintain repository cleanliness.
